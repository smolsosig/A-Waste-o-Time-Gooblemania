# A Waste o' Time: Gooblemania
![](web/assets/misc/gooble_logo.png)
## An offshoot of my variety platformer *A Waste o' Time*, hideous cat things have taken over Garcia Village... and of course it's your job to get rid of them all.

Play it on [Newgrounds](https://www.newgrounds.com/portal/view/1025974) and [itch.io](https://midnightsausage.itch.io/awot-gooblemania)!

**Requires Godot 4.6 and above.** Simply clone the repository and run it in Godot.
Note that there's lots of stuff in here done solely so the game can work on web, including weird hacky fixes for audio and all the desktop-only features from the main game getting stripped out.
